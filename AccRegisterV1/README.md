# # Features:
1. Basic Sign-Up system.
2. Basic Log-In system.
3. RAM temporary storage.

# # PROS:
1. Really lightweight.
2. Easy to test and highly suitable for prototype/foundation.
3. Clean and reusable code.

# # CONS:
1. There is no permanent data store.
2. Data is deleted after the program closes.
3. There is no text sanitization.
4. There is no security layer.
