# So What It Can Do?
- **Temporary RAM Storage:** Saves username and passwords on the RAM. No heavy databases here.
- **Fast Login Validation:** Utilizes Python's built-in dictionary methods to instantly cross-check credentials.

# How It Works?
Since Im only 3 days into Python, Im not an expert yet and havent dove into building fancy Graphic User Interface (GUI).
Instead, I kept it text-based and interactive! Here is how the logic flows:

- **The Core Loop:** The system boots up using a ```while True``` loop to keep a clean, text-based menu running continuously in the background.
- **Handling User Input:** I used Python built-in ```input()``` function to actively capture everything you type, whatever you are selecting a menu option, registering a name, or entering a password.
- **In-Memory Storage:** When logging in, ```input()``` collects your login attempts and instantly compares them againts the dictionary keys and values
  to grant or deny access.
